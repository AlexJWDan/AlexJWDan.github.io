Interests
Coding, Programming and Website Design

Coding is a slightly more recent hobby of mine. I started learning to code with C++ and eventually moved onto Python, along the way making various small projects/games such as a simple calculator and the game snake.

In the summer of 2020 I found a Facebook group that provided several live Javascript tutorials and decided to give the tutorial a go. This was my first introduction to website design and front end web development.
Personal Project

Store Locator

The store locator was the resulting product from a JavaScript tutorial taught by CleverProgrammer.com. It used a combination of HTML, CSS and JavaScript with the map provided by Google maps. The tutorial itself was a five-day JavaScript challenge in which the store locator app was created from scratch. I have to admit that it was a rather intense yet enriching experience that eventually resulted in the creation of this website.
The initial layout of the app was designed on diagrams.net. It was only a rough guide to aid the coding of the HTML and CSS and I therefore improved the design along the way.
This is a progress shot of the project before the search list and map were linked to the actual data with JavaScript code. This is why only one location point is visible.
Along with this I have added several animations though not all of them are visible any more as the Google API code has expired. But you are welcome to view it for yourself.
